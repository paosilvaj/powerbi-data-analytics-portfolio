# Sales Performance & Time Intelligence Dashboard (Power BI)

## 游섫릖 English

This project presents a **sales performance dashboard** built in Power BI, focused on tracking revenue, margins, and order volume over time.

The main objective is to analyze sales evolution using time-based comparisons (YTD, previous month, and previous year), supporting data-driven decision-making.

### Analysis scope
- Total sales and YTD sales
- Year-over-Year (YoY) and Month-over-Month (MoM) comparisons
- Total margin
- Order volume
- Analysis by region, category, and manager

### Data modeling
- **Star schema**
- Sales/orders fact table
- Calendar, region, and manager dimensions
- Dedicated measures table

### Key metrics (DAX)
- Total Sales
- Total Sales YTD
- Previous Year Sales
- Previous Month Sales
- Total Margin

### Tools used
- Power BI Desktop
- DAX (time intelligence)
- Data modeling
